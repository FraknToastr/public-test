About

Novel Lens

A self-contained browser prototype for compressed literary interpretation, evidence retrieval and historical-fork analysis.

Deterministic core

Vanilla HTML, CSS and JavaScript with an embedded copy of JSZip for EPUB archive reading. No CDN, tracking or network request is required.

Historical scope

The included historical corpus is intentionally compact and demonstrates the method. It is not a complete historical reference database.

Local AI

An optional OpenAI-compatible endpoint can challenge interpretations, but every response is grounded in retrieved passages and the generated dossier.
