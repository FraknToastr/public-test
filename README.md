# Event Calendar Dashboard v89 — GitHub Pages package

Deploy the contents of this folder to the root of a GitHub Pages branch or directory.

## Structure

- `index.html` — application markup
- `css/styles.css` — application styles
- `js/app.js` — application logic and embedded calendar data
- `assets/` — UOS light and dark SVG logos

Leaflet and PDF.js remain loaded from their existing public CDNs. The local CSS, JavaScript, and SVG references include `?v=89` cache-busting query strings.
